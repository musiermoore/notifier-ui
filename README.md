# notifier-ui

Vue SPA for the notifier web client.

Vue SPA для веб-клиента Notifier.

## Current User Actions

- Register and log in
- View current account profile
- Create notes and reminders
- Edit and delete items
- Filter items by all, reminders, notes, or recently changed
- Connect and disconnect Telegram
- Generate a Telegram link code and open the bot directly

- Регистрация и вход
- Просмотр текущего профиля
- Создание заметок и напоминаний
- Редактирование и удаление записей
- Фильтрация записей: все, напоминания, заметки, недавно измененные
- Подключение и отключение Telegram
- Генерация кода привязки Telegram и прямое открытие бота

## Language

- Default UI language: Russian
- Secondary language: English

- Язык интерфейса по умолчанию: русский
- Второй язык: английский

## Notes

- The UI is user-facing and should avoid development-only information.
- The desktop app can later reuse parts of this interface through Wails.

- Интерфейс ориентирован на обычного пользователя и не должен показывать лишнюю техническую информацию.
- В будущем части этого интерфейса можно переиспользовать в desktop-приложении через Wails.
